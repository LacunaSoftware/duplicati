Perguntas Frequentes

Como cadastro a aplicação?

É necessário um cadastro manual da aplicação nesta máquina para verificar que a mesma está autorizada a usar o serviço Backup e-notariado. Para cadastrar sua aplicação, execute os seguintes passos:
  • Copie o código acima, você pode selecionar todo o texto e copiar ou simplesmente clicar no ícone a direita
  • Visite o Portal E-Notariado Backup
  • Clique no botão "Aprovar computador"
  • Cole o código recém-copiado
  • Dê um nome ao computador ou use o que já está preenchido
  • De volta a esta página, aguarde a aplicação conferir o status automaticamente

O que fazer quando a aplicação não mostra a mensagem que está configurada corretamente e falha em se comunicar com o e-notariado?

A aplicação se comunica automaticamente com os servidores do e-notariado ao ser instalada. Caso acima não mostre a mensagem "A aplicação está configurada corretamente" e uma mensagem de erro sobre falha de comunicação apareça, houve algum problema nessa comunicação e será necessário refazê-la manualmente. Para isso clique em:

"Opção Avançada" > "Reiniciar identificação com e-notariado"
