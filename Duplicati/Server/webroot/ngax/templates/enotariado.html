Como cadastro a aplicação?

É necessário um cadastro manual da aplicação nesta máquina para verificar que a mesma está autorizada a usar o serviço Backup e-notariado. Para cadastrar sua aplicação, execute os seguintes passos:
  • Visite o Portal e-notariado Backup
  • Clique no botão "Cadastrar computador"
  • Aguarde que o portal detecte a instalação do Módulo Agente neste computador
  • Dê um nome ao computador ou use o que já está preenchido
  • De volta a esta página, aguarde a aplicação conferir o status automaticamente
